# Формат отправления

Поскольку система не позволяет загружать крейты, участникам доступны следующие билиотеки (обратите внимание на версии, они строго определены):

```toml
serde = "1.0.13"
serde_json = "1.0.13"
serde_derive = "1.0.13"
```

Решение представляет собой один или несколько rust-файлов, отправляемые `Cargo.toml` не учитываются.
